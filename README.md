
Este es el README del proyecto <b>LED CUBE</b>.

Se trata de un proyecto realizado para la asignatura de Electrónica Creativa, perteneciente al 4º curso del <i>Grado en Ingeniería de Sistemas Electrónicos</i> de la Universidad de Málaga.

En dicho proyecto, se implementa una librería para controlar cubos de nxnxn LEDs. Éste se completa a nivel hardware con un cubo de 8x8x8 LEDs.
